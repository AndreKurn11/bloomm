# Bloom Coffee & Place

Website café modern berbasis Laravel yang memungkinkan pelanggan melihat menu, memesan makanan dan minuman, melakukan checkout, serta mendapatkan receipt digital.

## Fitur

- Home Page
- Menu berdasarkan kategori
- Shopping Cart
- Checkout
- Receipt Digital
- Lokasi Café
- Halaman Karier

## Kategori Menu

- Coffee Specialities
- Non Coffee
- Mocktail & Squash
- Main Course
- Pasta
- Snack

## Teknologi

- Laravel 12
- PHP
- MySQL
- Bootstrap 5
- JavaScript
